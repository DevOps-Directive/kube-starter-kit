Extensions are components not included in the base kit but that you may want to add as your team and requirements grow.

<Note>
This section is under development. Check back for guides on adding these capabilities to your platform.
</Note>

## Planned Extensions

| Extension | Use Case |
|-----------|----------|
| **Service Mesh (Istio)** | mTLS, advanced traffic management, observability |
| **Datadog Integration** | Enterprise observability with APM |
| **Multi-region Setup** | Geographic redundancy and disaster recovery |
| **PlanetScale / RDS** | Managed database alternatives to CloudNativePG |
| **Vault** | Advanced secrets management with dynamic secrets |
| **VPC Peering** | Connect to other AWS accounts or on-premise |

Have a specific extension you'd like to see? [Open an issue](https://github.com/DevOps-Directive/kube-starter-kit/issues) on GitHub.
